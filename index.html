Filter Descriptions


Permit Types


Construction - A new structure


Demolition - Destroying an existing structure


Site Development - Property improvement without a new structure



Category Type


The general use category the permit falls into.


Commercial - Business or mixed use housing/commercial construction


Industrial – Heavy industry related construction


Institutional – Related to institutions such as schools or hospitals


Multifamily – Buildings which house multiple living units


Single Family/Duplex – Buildings with only one or two living units



Permit Status


How far along is the permitting process?


Application Accepted – The city has the application, but has not completed its review


Permit Issued – The city has issued a permit


Reviews Completed – The city has completed its review but has not issued a permit


Application Closed – The application has been withdrawn, rejected, or is otherwise out of consideration for permitting.



Retrieved Data


Address – The relevant property’s address

Applicant Name – The person who filed the application

Description – A brief description of the the proposed work. Long descriptions are truncated

Category – See category type above

Action Type – Is the application for a new building, changes to an existing building, or a demolition

Permit Number – The unique record number the city assigns

Permit Type – See permit type above

Value – Estimated worth of the finished construction based on fair market value.

Application Date – The date the application was filed

Issue Date – The date a permit was issued, if any

Expiration Date - The date an issued permit expires. Work should be completed

Permit and Complaints URL – A link to the City’s full record of the application
